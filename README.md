# Yew Blog

Rust で Web フロントを書いたブログです。  
SSG などはせず、GitHub の別リポジトリのマークダウンを読んで CSR してます。

### Note

1. Tailwind CSS のビルド

```
npm run build:talwindcss
```

2. ローカルサーバーの起動

```
trunk serve
```
